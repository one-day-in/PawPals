# Project Context

PawPals — веб-платформа, де власники собак/тварин створюють заявки на вигул, а виконавці (догвокери) беруть ці замовлення у роботу.

Бізнес-модель:
- Частково як фріланс-біржа (є пост замовлення + виконавець бере задачу)
- Частково як Uber-подібний потік замовлень (доступні заявки і оперативне взяття у виконання)

Поточний фокус MVP:
- Публічний список активних заявок
- Простий перегляд карток замовлень
- Mock API для швидкої розробки без бекенду
