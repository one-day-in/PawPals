# Typography Memory (dog walkers 3.pen)

- Font family: Inter (single typeface across project).
- Allowed font sizes: 8, 9, 10, 11, 12, 13, 14, 15, 16, 18, 20, 22, 24, 26.
- Allowed font weights: 400 (normal), 500, 600, 700, 800.
- Allowed line-heights: 1.4, 1.5, 1.6.
- Allowed letter-spacing: 0.3, 0.5.

## Hierarchy

- Display / key metrics: 20-26 with 700/800.
- Section headers: 14-16 with mostly 700.
- Body text: 11-12 with 400/500.
- Meta labels / chips: 8-10 with 500/600.

## Navbar Tokens

- Logo: 18 / 700 / blue-600.
- Menu items: 13 / 500 / font-secondary.
- Search placeholder: 12 / 400 / dark-400.
- CTA buttons: 13 / 600 (white text for primary).

## Text Color Priority

- font-primary, font-secondary, font-tertiary, blue-600, white.
- Spot accents: green-500, amber-500, dark-300, dark-400, red-500, star-color.
